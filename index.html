Chronicles

Chronicles is a fully responsive website which explores different eras throughout surfing's history. A draggable jQuery UI slider is implemented as a means of navigating through the eras. The content for each era is loaded asynchronously using Ajax. Each event is displayed using a Bootstrap modal. The website makes use of fullscreen HTML5 video as a background.

SKATE

SKATE is a programme which lets users search skate spots to shred where they're able to add, edit, search, review and comment on spots around the globe. Spots can be defined as anything from a giant rail, a smooth curve, a long windy hill or an actual park in which they're catogorised into ability levels. I created this application to get a grasp on backend development/databases. Technologies I used were Node.js, Express and MongoDB
